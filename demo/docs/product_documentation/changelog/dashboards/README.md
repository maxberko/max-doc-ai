# Dashboards Release

## Overview

**Feature:** Dashboards
**Category:** Features
**Release Date:** TBD
**Target Audience:** All FlowState users, especially team leads and managers

## Description

Dashboards give teams instant visibility into workflow performance with customizable, real-time views. Users can track metrics, spot trends, and make data-driven decisions without digging through logs or running manual reports.

## Key Capabilities

- Real-time workflow metrics and performance tracking
- Drag-and-drop dashboard customization
- Smart insights and recommendations
- Team collaboration and sharing
- Export and reporting capabilities
